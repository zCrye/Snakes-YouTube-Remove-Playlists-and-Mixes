# Snakes YouTube Remove Playlists and Mixes

Chrome extension that removes:
- Playlists & playlist panels
- Mixes / Auto-generated radio
- Related playlist suggestions on watch pages

## Features
- Cleans URLs automatically
- Removes playlist sidebar
- Toggle on/off via popup
- Works on all YouTube watch pages

## Installation
1. Download or clone this repo
2. Go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select this folder
